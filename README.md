## 🌍 Overview

**Ashes of Velhara** is a work-in-progress open-world survival game focused on exploration, resource management, and immersive environment design.

This project is part of my game development portfolio, where I’m building and improving core gameplay systems using Unreal Engine 5.

---

## 🚀 Features (Current & Planned)

* Open-world exploration
* Player movement and interaction system
* Basic survival mechanics (health, stamina)
* Environment blockout and level design

---

## 🛠 Tech Stack

* **Engine:** Unreal Engine 5 (Blueprints, basic C++)
* **Programming:** C++ (learning), Blueprint scripting
* **Tools:** Visual Studio, Blender
* **Version Control:** Git
